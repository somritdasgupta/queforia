## Overview

A premium brand experience landing built on top of NextJS and TailwindCSS.

## Tech Stack

- Framework: Next.js 14
- Styling: TailwindCSS
- Animation: Framer Motion
- Icons: Lucide Icons
- Fonts: Inter, Prompt
- Runtime: Edge

## Features

- Next.js 14 App Router
- Responsive design
- Dark mode optimized
- PWA enabled
- Dynamic OG images
- Brand Kit playground
